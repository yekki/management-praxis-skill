# SaaS / 软件行业模板

## 常见场景与推荐工具链

### 场景 1：新市场进入
**问题**：是否进入某个新市场/新区域？
```
PEST(1.7) → 波特五力(1.8) → EFE矩阵(1.1) → SWOT(1.3) → 国际化进入战略(1.16)
```
**重点关注**：数据合规(P)、付费意愿(E)、SaaS接受度(T)、替代品强度

### 场景 2：竞争定位
**问题**：和竞品怎么差异化？怎么定价？
```
波特五力(1.8) → 价值链分析(1.11) → 基本竞争战略(1.9) → 市场细分(8.4) → 四象限定位(8.5)
```
**重点关注**：SaaS产品功能同质化严重，差异化往往在服务和生态而非功能

### 场景 3：产品创新
**问题**：下一个版本做什么功能？怎么创新？
```
缺点列举法(3.4) → 5W2H(3.1) → 头脑风暴(3.7) → KJ法(3.10) → 产品生命周期(8.6)
```
**重点关注**：SaaS产品要平衡创新与稳定，避免功能膨胀

### 场景 4：增长瓶颈
**问题**：增长放缓怎么办？
```
产品/市场扩展方格图(8.8) → BCG矩阵(1.4) → 价值链分析(1.11) → CLV顾客终身价值(9.5) → 一对一营销(9.17)
```
**重点关注**：SaaS的增长三杠杆——获客、留存、扩展收入（upsell/cross-sell）

### 场景 5：定价策略
**问题**：怎么定价？免费版要不要做？
```
成本加成定价(8.10) → 认知价值定价(8.12) → 动态定价(8.15) → 顾客价值等式(9.4)
```
**重点关注**：SaaS定价核心是价值定价，不是成本定价；Freemium模式的转化率基准

### 场景 6：客户流失
**问题**：客户续费率低怎么办？
```
SERVQUAL(9.15) → 服务质量差距模型(9.9) → 抱怨冰山(9.6) → 满意镜(9.7) → CCPR(9.23)
```
**重点关注**：SaaS的churn分析要区分logo churn和revenue churn

## SaaS行业特有注意事项

1. **指标体系**：ARR/MRR、CAC、LTV、Churn Rate、NRR 是核心指标，分析时优先关联
2. **网络效应**：SaaS产品是否有网络效应决定了护城河深度，波特五力中要特别评估
3. **切换成本**：数据迁移难度是客户锁定的关键，买方议价力分析时重点考虑
4. **PLG vs SLG**：产品驱动增长 vs 销售驱动增长，影响渠道和营销工具的选择
