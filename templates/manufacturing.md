# 制造业模板

## 常见场景与推荐工具链

### 场景 1：质量改进
**问题**：产品质量不稳定、客诉多
```
分层法(5.10) → 排列图(5.11) → 因果图(5.12) → PDCA(5.9) → 控制图(5.15) → 6σ(4.1)
```
**重点关注**：先用排列图找到关键少数问题，再用因果图深挖根因，不要撒网式改进

### 场景 2：精益转型
**问题**：生产效率低、浪费多、库存高
```
价值链分析(1.11) → 精益生产(4.11) → JIT(4.10) → TPM(4.9) → 现场5S(4.8)
```
**重点关注**：精益不只是工具，是管理哲学；从5S起步建立基础纪律

### 场景 3：供应链优化
**问题**：供应链成本高、交期不稳
```
SCM(4.13) → MRP(4.14) → 闭环MRP(4.15) → MRPII(4.16) → ABC分析(5.7)
```
**重点关注**：ABC分类管理库存，A类物料重点管控

### 场景 4：战略规划
**问题**：制造业企业的战略方向选择
```
PEST(1.7) → 波特五力(1.8) → 企业核心能力识别(1.12) → SWOT(1.3) → QSPM(1.6)
```
**重点关注**：制造业核心能力通常在工艺、成本控制、供应链整合

### 场景 5：新产品开发
**问题**：从概念到量产
```
5W2H(3.1) → 质量机能展开QFD(9.13) → 甘特图(5.2) → 关键路线法(5.3) → 工作分解结构(5.4)
```
**重点关注**：QFD确保客户需求转化为产品规格，关键路线法管控开发进度

### 场景 6：人效提升
**问题**：人工成本上升、效率待提升
```
工作写实法(6.1) → 测时法(6.2) → 工作抽样法(6.3) → 因素比较法(6.8) → 图尺度考评法(6.21)
```
**重点关注**：先测量再改进，用数据说话

## 制造业特有注意事项

1. **安全第一**：任何改进方案都必须评估安全影响，ISO体系(4.4/4.5)是底线
2. **设备OEE**：TPM(4.9)关注设备综合效率，是制造业特有的核心指标
3. **现场管理**：制造业改进从现场（Gemba）开始，5S(4.8)是基础
4. **标准化**：改进成果必须固化为标准作业程序，否则会回退
