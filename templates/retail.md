# 零售 / 消费行业模板

## 常见场景与推荐工具链

### 场景 1：市场定位与品牌策略
**问题**：品牌定位不清、目标客群模糊
```
市场细分(8.4) → 四象限定位(8.5) → 品牌/投入模式(8.2) → CIS企业形象策划(4.19) → CS战略(4.20)
```
**重点关注**：零售的核心是"卖给谁"比"卖什么"更重要

### 场景 2：渠道与增长
**问题**：线上线下怎么选？渠道怎么布局？
```
产品/市场扩展方格图(8.8) → 渠道选择经济评估(8.16) → 特许经营(8.18) → 连锁经营(8.19) → 推拉战略(8.20)
```
**重点关注**：全渠道不等于每个渠道都做，算清每个渠道的单位经济模型

### 场景 3：定价与促销
**问题**：怎么定价？促销预算怎么分？
```
成本加成定价(8.10) → 认知价值定价(8.12) → 动态定价(8.15) → 价格/促销方格图(8.9) → 促销预算法(8.21)
```
**重点关注**：消费品定价要兼顾品牌调性，频繁促销会伤害品牌

### 场景 4：客户留存
**问题**：复购率低、会员流失
```
CLV顾客终身价值(9.5) → 一对一营销(9.17) → 关系营销(9.18) → RATER指数(9.16) → CCPR(9.23)
```
**重点关注**：零售业CLV计算要包含口碑推荐价值

### 场景 5：门店运营优化
**问题**：门店坪效低、客流下降
```
ABC分析(5.7) → 现场5S(4.8) → 服务让渡系统(9.3) → 满意镜(9.7) → 能力循环(9.2)
```
**重点关注**：门店体验=商品×环境×人员，三者缺一不可

### 场景 6：供应链与库存
**问题**：库存周转慢、缺货与积压并存
```
ABC分析(5.7) → SCM(4.13) → MRP(4.14) → JIT(4.10) → 滚动计划法(5.1)
```
**重点关注**：快消品和耐用品的库存策略完全不同

## 零售行业特有注意事项

1. **季节性**：零售分析必须考虑季节波动，同比优于环比
2. **坪效/人效**：零售核心效率指标，所有改进最终要回到这两个数
3. **消费者洞察**：市场调研工具(8.24-8.27)在零售业使用频率最高
4. **快速迭代**：零售市场变化快，PDCA(5.9)循环要比制造业更短
