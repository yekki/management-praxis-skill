# 专业服务行业模板

## 常见场景与推荐工具链

### 场景 1：服务质量体系搭建
**问题**：服务标准不统一、客户满意度参差
```
SERVQUAL(9.15) → 服务质量差距模型(9.9) → 服务质量控制(9.11) → 波卡纠偏(9.12) → ISO9000(4.4)
```
**重点关注**：专业服务的质量高度依赖人，标准化和个性化之间要平衡

### 场景 2：客户关系管理
**问题**：大客户依赖度高、客户拓展困难
```
CRM(4.21) → CLV(9.5) → 关系营销(9.18) → IMIM过程(9.20) → 一对一营销(9.17)
```
**重点关注**：专业服务80%收入来自20%客户，关系维护是命脉

### 场景 3：团队能力建设
**问题**：人才是核心资产，怎么选育用留？
```
胜任特征评价(6.12) → 评价中心法(6.10) → 360度反馈(6.9) → 管理案例法(6.26) → 工作教练法(6.27)
```
**重点关注**：专业服务的培训投入回报最高，学徒制（工作教练法）效果优于课堂

### 场景 4：战略增长
**问题**：业务增长靠什么驱动？
```
企业核心能力识别(1.12) → 价值链分析(1.11) → 产品/市场扩展方格图(8.8) → 服务利润链(9.1) → 标杆管理(4.23)
```
**重点关注**：专业服务增长三路径——深耕客户、扩品类、复制模式

### 场景 5：定价与项目管理
**问题**：项目报价、成本控制、交付管理
```
认知价值定价(8.12) → 甘特图(5.2) → 关键路线法(5.3) → 工作分解结构(5.4) → 目标管理(4.22)
```
**重点关注**：专业服务定价基于价值而非工时，但成本管控仍需WBS拆解

## 专业服务行业特有注意事项

1. **人即产品**：员工满意度直接等于客户满意度（满意镜9.7），人力资源工具使用频率最高
2. **知识管理**：知识管理(4.24)是专业服务企业的核心竞争力基础
3. **信任经济**：服务金三角(9.19)中，"人"的权重远大于"系统"
4. **利用率**：顾问/律师的计费利用率是核心运营指标
