# 问题诊断方法论

## 问题分类矩阵

| 维度 | 战略层 | 运营层 | 人员层 |
|------|--------|--------|--------|
| **方向性问题** | 市场定位不清、战略缺失 | 流程混乱、标准缺失 | 角色不明、职责不清 |
| **效率问题** | 资源配置低效、业务组合失衡 | 产能不足、质量不稳 | 绩效低下、能力不足 |
| **增长问题** | 增长瓶颈、新市场进入 | 产品创新乏力、渠道不足 | 人才梯队断层 |
| **危机问题** | 竞争威胁、行业变革 | 供应链断裂、客户流失 | 核心人才流失、士气低落 |

## 诊断路径

### 第一步：定性判断
- 这是"不知道做什么"（方向问题）还是"不知道怎么做"（方法问题）？
- 这是"没做到"（执行问题）还是"做错了"（决策问题）？

### 第二步：定位层次
- 公司级：影响整体战略方向和资源配置
- 部门级：影响某个职能的效率和产出
- 项目级：影响具体任务的执行和结果

### 第三步：匹配工具域
| 问题定位 | 首选工具域 | 辅助工具域 |
|----------|-----------|-----------|
| 战略方向 | 战略管理(ch1) | 领导与组织(ch2) |
| 竞争应对 | 战略管理(ch1) | 营销(ch8) |
| 产品创新 | 创新(ch3) | 营销(ch8) |
| 流程优化 | 管理体系(ch4) | 生产及质量(ch5) |
| 人员管理 | 人力资源(ch6) | 领导与组织(ch2) |
| 财务决策 | 财务管理(ch7) | 战略管理(ch1) |
| 市场拓展 | 营销(ch8) | 战略管理(ch1) |
| 服务提升 | 客户服务(ch9) | 营销(ch8) |
