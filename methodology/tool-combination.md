# 工具组合策略

## 组合原则

1. **输入-输出衔接**：上一个工具的输出，是下一个工具的输入
2. **宏观到微观**：先用宏观工具定方向，再用微观工具定细节
3. **分析到决策**：先用分析工具理解现状，再用决策工具选择方案
4. **定性到定量**：先定性判断框架，再定量评估排序

## 经典工具链

### 1. 战略规划全链条
```
PEST（宏观环境）
  → EFE矩阵（外部机会威胁量化）
  → 波特五力（行业竞争格局）
  → IFE矩阵（内部优劣势量化）
  → SWOT分析（内外交叉匹配）
  → QSPM（战略方案定量排序）
  → 平衡计分卡（战略目标分解）
```

### 2. 产品创新链条
```
缺点列举法/希望点列举法（发现改进方向）
  → 5W2H（结构化问题定义）
  → 头脑风暴/635法（创意发散）
  → KJ法（创意分类归纳）
  → 形态分析法（方案系统组合）
  → 产品生命周期（市场时机判断）
```

### 3. 质量改进链条
```
分层法（数据分类）
  → 排列图/帕累托（找关键问题）
  → 因果图/鱼骨图（分析根因）
  → PDCA循环（改进执行）
  → 控制图（效果监控）
  → 6σ/TQM（体系化巩固）
```

### 4. 人才管理链条
```
工作分析（职位问卷/面谈法）
  → 评价中心法/胜任特征（人才评估）
  → 360度反馈（多维反馈）
  → 平衡计分卡+KPI（绩效管理）
  → 培训体系（案例法/教练法/角色扮演）
```

### 5. 市场进入链条
```
PEST（目标市场宏观环境）
  → 市场细分法（选定目标客群）
  → 四象限定位法（竞争定位）
  → 产品/市场扩展方格图（增长路径）
  → 定价策略选择（成本/价值/竞争导向）
  → 渠道评估（经济评估法）
```

### 6. 客户服务提升链条
```
SERVQUAL/RATER指数（服务质量评估）
  → 服务质量差距模型（定位差距来源）
  → 抱怨冰山（理解投诉背后的真实不满）
  → 满意镜（员工-客户满意度联动）
  → 服务利润链4P+3R（建立系统）
  → CCPR法（落地执行）
```

## 跨域组合模式

| 组合模式 | 涉及领域 | 适用场景 |
|----------|---------|---------|
| 战略+财务 | ch1+ch7 | 投资决策、并购评估 |
| 战略+营销 | ch1+ch8 | 新市场进入、竞争战略落地 |
| 创新+生产 | ch3+ch5 | 新产品开发到量产 |
| HR+领导 | ch6+ch2 | 组织变革、文化建设 |
| 营销+服务 | ch8+ch9 | 客户全生命周期管理 |
| 体系+生产+HR | ch4+ch5+ch6 | 精益转型、全面质量管理 |
